Number Cakes

Hyper tendance, nos Number Cakes se déclinent sous différentes formes : chiffres, lettres, cercle, cœur, ou toute autre forme selon vos envies.

Tarifs

À partir de 5 € la part (hors personnalisation). Les décorations spécifiques et les impressions alimentaires sont proposées en supplément.

Personnalisation

Pour toute demande de devis, merci d’indiquer :

  • le thème ;
  • le modèle (forme souhaitée) ;
  • le goût et la base (génoise nature, chocolat ou sablée) ;
  • la crème (chantilly mascarpone, ganache, crème diplomate…) ;
  • les suppléments gourmands : speculoos, caramel beurre salé, fruits rouges, etc.

Portions disponibles

Commandes possibles pour 8, 12, 15 parts, et selon certaines formes, au-delà.
